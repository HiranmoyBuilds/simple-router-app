This is a starters app in node.js to route various paths & perform crud operations using MVC architecture.

This basic app is locally hosted and can handle the tasks as listed below: 
1.Register new user in Mongo database.
2.Get the list of all the users registered.
3.Update the users details(dynamically in the future updates).
4.Delete the users.
5.Create a log file to keep track of time,ip, method and path of the request made to the server.

Post, delete and other methods are being handled by Postman utility server as of now.
